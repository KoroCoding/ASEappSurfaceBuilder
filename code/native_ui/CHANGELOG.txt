ASEapp Surface Builder CHANGELOG
================================

Unreleased
----------

Version 1.3.0 - Matrix supercell transforms
------------------------------------------------
- Added optional direct 3x3 integer lattice-transform input to the Supercell dialog, so rotated supercells such as sqrt(2) x sqrt(2) R45 can be generated without adding one-off presets.
- Kept the existing a/b/c multiplier workflow as the default, while showing determinant and expected atom count validation in matrix mode before Apply.
- Preserved matrix-based supercell state through snapshots, undo/redo, and tab switching.
- Added a GUI self-test assertion for the sqrt(2) x sqrt(2) R45 transform, including determinant atom count, rotated cell lengths, orthogonality, and fractional-coordinate bounds.
- Updated the operation guide and packaged quick-start text for the new matrix-input workflow.

Version 1.2.0 - VESTA-like display and startup polish
------------------------------------------------------
- Added a side preview to the Atom legend PNG export dialog, plus label prefix/suffix fields and manual element-order controls.
- Cached the extracted standalone payload under the user's local app-data folder using an embedded-payload hash, so repeat launches skip the slow ZIP extraction path.
- Replaced string-formatted spatial grid keys in atom-neighbor searches with compact integer keys to reduce allocation overhead for larger structures.
- Added an immediate standalone-launcher splash screen that stays visible during embedded ZIP extraction and fades out when the native UI starts.
- Increased the VESTA-like bond stroke width while keeping one-piece, constant-width two-color bond rendering.
- Ignored sub-0.50 A symmetry near-overlap artifacts for display density and bond rendering so the magnemite CIF keeps normal atom sizes.
- Adjusted the default ball-and-stick atom scale and fit radius to match VESTA more closely; the initial fit now prioritizes visible atoms instead of forcing the full cell box on screen.
- Changed normal bonds to VESTA-style half-and-half coloring using the colors of the atoms at each end.
- Added the Outside cell toggle to show/hide VESTA-like periodic image atoms and bonds crossing the unit-cell boundary.
- Added a GUI self-test assertion that the Outside cell toggle adds periodic boundary bonds.
- Fixed the missing Save As path for CIF export; current structures can now be written as P1 CIF files.
- Regenerated the verified Windows standalone launcher and macOS DMG as v1.2.0.

Version 1.1.2 - Refactor and verified Windows rebuild
------------------------------------------------------
- 起動時の Qt plugin path 設定を AppStartup に分離し、main.cpp を起動フローへ集中させました。
- 単一起動と既存ウィンドウへのファイル転送を SingleInstance に分離しました。
- Windows standalone launcher の HANDLE / LocalFree 管理を RAII 化し、機能を変えずに解放経路を単純化しました。
- VERSIONING.md を追加し、今後の SemVer ベースの番号付け基準を明文化しました。
- Windows 単体 launcher / ZIP を v1.1.2 として再生成し、Release build、CTest、起動確認、署名確認を実施しました。

- CIF 読み込み時に `_space_group_symop_operation_xyz` / `_symmetry_equiv_pos_as_xyz` の対称操作を展開し、代表点だけでなく単位胞内の原子を描画するように修正。
- CIF のインラインコメントを quote-aware に無視し、セル値や対称操作行の末尾コメントで読み込みが崩れないように修正。
- VASP/POSCAR 保存で Direct / Cartesian を選択できるように変更。
- 通常 XYZ 保存で Cartesian / 分率座標を選択できるように変更。extended XYZ は ASE 互換の Cartesian `pos` を維持。
- Windows で2回目以降の起動に渡したファイルパスを既存ウィンドウへ送れるように修正。
- Windows 起動時に Qt plugin path を実行ファイル周辺へ固定し、`no Qt platform plugin could be initialized` ダイアログが繰り返し出ないように修正。
- README / Guide / QUICKSTART を CIF 対称性展開と保存座標形式に合わせて更新。

Version 1.1.1 - Single-instance tabs and atom-pair bond editing
---------------------------------------------------------------
- アプリを単一起動に制限し、2回目以降の起動では新しいウィンドウを作らず既存ウィンドウを前面へ戻すように変更。
- 構造タブを追加し、1つのウィンドウ内で複数構造を切り替えられるように変更。
- Open は既定で新しいタブに読み込み、未編集の初期タブだけは置き換えるように変更。
- 原子対ボンド長ダイアログを追加し、読み込まれている構造の全原子対距離を一覧・個別編集できるように変更。
- README / QUICKSTART / 配布手順を v1.1.1 向けに更新。

Version 1.1.0 - Adsorbate pose editing release
----------------------------------------------
- 吸着分子ポーズ編集を追加し、選択原子群を剛体グループとして並進・回転できるように変更。
- pivot 原子固定回転、選択中 2 原子の結合軸回転、選択結合長調整を追加。
- 並進 / 回転 / 結合長調整のポーズプレビューを追加。
- 2 原子間 / 3 原子間などの配置候補を、選択原子すべての中心へ統一。
- VESTA SBOND 風の元素ペア別ボンド距離 min/max Å 設定を追加。
- extended XYZ / adsorbate-only XYZ / pose JSON / ASE snippet 出力を追加。
- Undo / Redo が構造、選択、吸着分子グループ情報を同期して復元するように修正。
- トラックパッドのピンチズームを、ホイールと同じくカーソル位置基準で使えるように修正。
- メタノール on Cu 実分子サンプルの GUI 自己テストと ASE round-trip テストを追加。
