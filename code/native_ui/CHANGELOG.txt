ASEapp Surface Builder CHANGELOG
================================

Version 1.0.0 - Final local release
-----------------------------------
- Windows 向け Qt ネイティブ UI を配布可能な ZIP / 単体 launcher として整備。
- POSCAR / CONTCAR / .vasp / .xyz / .cif / .pdb / .xsf / .json / .aseproj / .vesta 読み込みに対応。
- VESTA 風の原子表示、視点操作、原子選択、重なり選択を整備。
- Ctrl + 左ドラッグで奥に重なった原子も選択可能に変更。
- Esc で選択解除可能に変更。
- 複数選択時の直上/直下配置を、選択全原子へ一括適用。
- 前駆体CSV保存/読込/配置を追加。
- 周期表から生成元素を選択可能に変更。
- 周期表は一覧を小型カード化し、詳細情報はホバー時の暗色ポップアップへ集約。
- 周期表ダイアログは説明文・凡例・下部ボタンを省き、周期表全体を優先表示。
- Supercell、真空層除去、セル軸傾き、slab 全体移動を整備。
- 原子一覧PNG出力を追加し、解像度/DPI/背景/列数/原子数表示を選択可能に変更。
- 遠景や Supercell 表示時の原子球サイズが不自然に揃わないよう調整。
- アイコンと Windows launcher 配布を整備。
- 文字なし・透過背景の分子ネットワーク風アイコンへ更新。
- アイコン元ファイルを SVG 1 つに統一し、Windows ICO / PNG / macOS ICNS はビルド時生成に変更。
- README / QUICKSTART / 配布手順を整理。
