ASEapp Surface Builder クイックスタート
======================================

1. 起動
------
- 単体版: ASEappSurfaceBuilder-1.0.0-Windows.exe を実行します。
- 展開版: bin\ASEappNativeUI.exe を実行します。

2. 構造を開く
------------
- Open を押すか、構造ファイルを画面へドラッグ&ドロップします。
- POSCAR / CONTCAR / .vasp / .xyz / .cif / .pdb / .xsf / .json / .aseproj / .vesta に対応しています。

3. 原子を選択する
----------------
- 左クリック: 原子を選択
- Ctrl + 左クリック: 選択追加/解除
- Ctrl + 左ドラッグ: 重なった奥の原子も追加選択
- Esc: 選択解除

4. 原子を置く
------------
- 周期表ボタンで元素を選びます。
- 配置位置を「直上」「直下」「原子間」「多点中心」などから選びます。
- Apply を押します。
- 複数選択時の直上/直下は、選択された全原子に一括適用されます。

5. 前駆体を使う
--------------
- 前駆体にしたい原子群を選択します。
- 前駆体CSV保存で名称を入力して保存します。
- 前駆体CSV読込で読み込むと、名称一覧が前駆体ドロップダウンに表示されます。
- 生成元素とは別に、配置したい前駆体名を前駆体ドロップダウンで選びます。
- 通常の原子配置と同じ「配置位置」を選び、前駆体配置でその位置へまとめて配置します。
  前駆体内で最も低い原子が配置先の原点になります。

6. slab を整える
---------------
- Supercell で a/b/c 方向へ拡張します。
- 真空層除去で c 軸方向の余分な真空を詰めます。
- Supercell 後に真空層除去しても、倍率表示は 1x1x1 に戻りません。
- セル軸傾きで c 軸を a/b 方向へ傾け、step-terrace 候補を作れます。
- slab 全体移動で a/b/c 方向へ位置を調整できます。

7. 発表用画像を出す
------------------
- 原子一覧PNG で、構造内元素の球＋ラベル画像を出力します。
- 論文・学会用には 2000 px 以上または 300 dpi 以上を推奨します。

8. 保存
------
- Save で POSCAR 形式として保存します。
- 重要な構造は、編集前に別名保存しておくと安全です。
