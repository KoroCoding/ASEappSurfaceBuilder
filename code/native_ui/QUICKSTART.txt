ASEapp Surface Builder クイックスタート
======================================

1. 起動
------
- 単体版: ASEappSurfaceBuilder-1.2.0-Windows.exe を実行します。
- 展開版: bin\ASEappNativeUI.exe を実行します。
- macOS DMG 版: ASEappNativeUI.app を Applications へドラッグし、右クリックまたは Control + クリックで「開く」を選びます。

2. 構造を開く
------------
- Open を押すか、構造ファイルを画面へドラッグ&ドロップします。
- 2回目以降にアプリを起動しても別ウィンドウは増えず、既存ウィンドウを前面へ戻します。
- Ctrl + T で空のタブを追加、Ctrl + W で現在のタブを閉じます。
- POSCAR / CONTCAR / .vasp / .xyz / .extxyz / .cif / .pdb / .xsf / .json / .aseproj / .vesta に対応しています。
- CIF は代表的な対称操作ループを展開して、非対称単位だけでなく単位胞内の原子を描画します。
- ホイールまたはトラックパッドのピンチで、カーソル位置を基準にズームできます。
- 初期表示は VESTA 風の ball-and-stick に近づけるため、原子ラベルはオフです。必要ならツールバーの「ラベル」で表示できます。
- ボンドは両端原子の色で半分ずつ、端から中心まで同じ太さで表示されます。
- ツールバーまたは表示メニューの「格子外」で、VESTA風に周期境界で単位格子外へ続く複製原子とボンドを表示/非表示できます。

3. 原子を選択する
----------------
- 左クリック: 原子を選択
- Ctrl + 左クリック: 選択追加/解除
- Ctrl + 左ドラッグ: 重なった奥の原子も追加選択
- Esc: 選択解除

4. 原子を置く
------------
- 周期表ボタンで元素を選びます。
- 配置位置を「直上」「直下」「原子間」「多点中心」などから選びます。
- Apply を押します。
- 複数選択時の直上/直下は、選択された全原子に一括適用されます。

5. 前駆体を使う
--------------
- 前駆体にしたい原子群を選択します。
- 前駆体CSV保存で名称を入力して保存します。
- 前駆体CSV読込で読み込むと、名称一覧が前駆体ドロップダウンに表示されます。
- 生成元素とは別に、配置したい前駆体名を前駆体ドロップダウンで選びます。
- 通常の原子配置と同じ「配置位置」を選び、前駆体配置でその位置へまとめて配置します。
  前駆体内で最も低い原子が配置先の原点になります。

6. 吸着分子ポーズを調整する
--------------------------
- 吸着分子の原子群を選択し、吸着分子ポーズ編集でグループ化します。
- 並進 / 回転 / 結合長の数値を入力します。
- 必要な時だけ「ポーズプレビューを表示」をオンにし、プレビュー対象を選んで Apply 後の半透明位置を確認します。
- 問題なければ「並進を適用」「回転を適用」「結合長を適用」を押します。
- ASE に戻す場合は XYZ/extXYZ、pose JSON、ASE snippet を出力します。

7. 原子対ボンド長を調整する
----------------------------
- ツールバーの「原子対ボンド長」を開きます。
- 読み込まれている構造の全原子対が距離順に表示されます。
- 修正したい行の Target Å を変更し、「変更済みをすべて適用」または「選択行を適用」を押します。
- Atom A は固定され、Atom B が現在の原子対方向に沿って移動します。

8. slab を整える
---------------
- Supercell で a/b/c 方向へ拡張します。
- 真空層除去で c 軸方向の余分な真空を詰めます。
- Supercell 後に真空層除去しても、倍率表示は 1x1x1 に戻りません。
- セル軸傾きで c 軸を a/b 方向へ傾け、step-terrace 候補を作れます。
- slab 全体移動で a/b/c 方向へ位置を調整できます。

9. 発表用画像を出す
------------------
- 原子一覧PNG で、構造内元素の球＋ラベル画像を出力します。
- 論文・学会用には 2000 px 以上または 300 dpi 以上を推奨します。

10. 保存
------
- Save で CIF / POSCAR / extended XYZ / 通常 XYZ 形式として保存します。
- CIF は現在表示中の全原子とセルを P1 の構造として書き出します。
- POSCAR は Direct / Cartesian、通常 XYZ は Cartesian / 分率座標を選択できます。
- 重要な構造は、編集前に別名保存しておくと安全です。
- 原子一覧PNGは、横プレビュー、Prefix/Suffix、表示順リストを確認してから保存します。
