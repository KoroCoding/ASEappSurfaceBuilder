# ASEapp Surface Builder v1.1.1 リリースノート

リリース日: 2026-05-14
対象: Windows 配布版 `ASEappSurfaceBuilder-1.1.1-Windows.exe`

## 概要

v1.1.1 は、1つの ASEapp Surface Builder ウィンドウで複数構造を安全に扱うためのアップデートです。アプリの多重起動を防ぎ、2回目以降の起動では既存ウィンドウを前面へ戻します。あわせて構造タブと、読み込み済み構造の全原子対距離を一覧・個別編集できる原子対ボンド長機能を追加しました。

## 主な追加機能

### 単一起動

- `ASEappSurfaceBuilder` の多重起動を抑止します。
- 2回目以降の起動は新しいウィンドウを作らず、既存ウィンドウを前面へ戻します。
- 複数構造は、既存ウィンドウ内の `Open`、ドラッグ&ドロップ、新規タブから扱います。

### 構造タブ

- 1つのウィンドウ内で複数構造をタブ切り替えできます。
- `Open` は既定で新しいタブとして構造を開きます。
- 未編集の初期タブだけは、最初の読み込み時に置き換えます。
- `Ctrl + T` で空タブを追加、`Ctrl + W` で現在タブを閉じられます。
- 未保存変更があるタブを閉じる時は確認します。

### 原子対ボンド長一覧・編集

- ツールバーの「原子対ボンド長」から、読み込まれている構造の全原子対距離を一覧できます。
- 現在距離と Target Å を行ごとに表示します。
- Target Å を変更し、「変更済みをすべて適用」または「選択行を適用」で個別に反映できます。
- Atom A は固定され、Atom B が現在の原子対方向へ移動します。

## 修正・改善

- タブごとに構造、選択状態、Undo / Redo、吸着分子ポーズ、読み込み済み前駆体を保持するようにしました。
- 言語切替後も現在タブとタブ一覧を復元するようにしました。
- README、QUICKSTART、配布手順、同梱 CHANGELOG を v1.1.1 向けに更新しました。

## 配布物

- Windows 単体版: `ASEappSurfaceBuilder-1.1.1-Windows.exe`
- Windows ZIP 展開版: `ASEappSurfaceBuilder-1.1.1-Windows.zip`
- Windows 旧版 v1.1.0 / v1.0.0 は互換確認用として `standalone_exe/windows/` に残しています。
- macOS 版は現時点では v1.1.0 が同梱版です。v1.1.1 は macOS 環境で再パッケージしてください。

## 注意点

- 原子対ボンド長編集は全原子対を対象にするため、非常に大きな構造では一覧表示に時間がかかる場合があります。
- 原子対ボンド長編集は、指定した Atom A を固定し Atom B のみを移動します。分子全体を剛体として動かしたい場合は、従来の吸着分子ポーズ編集を使ってください。
- 自己署名はローカル開発確認向けです。組織管理 PC や Smart App Control 環境では、信頼済み署名が必要になる場合があります。
